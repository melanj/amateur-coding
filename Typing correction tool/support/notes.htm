1. Installing Easy Sinhala Toolkit

This version of Easy Sinhala Toolkit for Microsoft Windows XP. Linux and Other Unix versions not available yet. Download setup.zip from
http://www.ums.tripod.com/product/estoolkit/, then Extract zip file and run setup.exe The setup program will install Easy Sinhala Toolkit
and Windows XP Hotfix - Sinhalese Enabling Pack

2. Enabling complex Script Support

  1. Open Regional and Language Options in Control Panel.
  2. On the Languages tab, under Supplemental language support, select the Install files for complex script and right-to-left languages check box.
  3. Click OK or Apply.
  4. After the files are installed, you must restart your computer.

Notes

  • To open Regional and Language Options, click Start, click Control Panel, and then double-click Regional and Language Options.
  • You must be logged on as an administrator or a member of the Administrators group in order to complete this procedure. If your computer is connected to a network, network policy settings might also prevent you from completing this procedure.

3. Adding the Sinhala keyboard

  1. Open Regional and Language Options in Control Panel.
  2. On the Languages tab, under Text services and input languages, click Details.
  3. On the Settings tab, Under Installed services, click Add.
  4. In the Input language list, Select an input language "Sinhala (Sri Lanka)”.
  5. select the Keyboard layout/IME check box, and then Select a keyboard layout "Sinhala" or "Sinhala - Wij 9" in the list.
  6. Click OK
  7. Click OK or Apply to close Regional and Language Options.

4. Customize key sequences for languages

  1. Open Regional and Language Options in Control Panel.
  2. On the Languages tab, under Text services and input languages, click Details.
  3. Under Preferences, click Key Settings.
  4. In the Advanced Key Settings dialog box, under Hot keys for input languages, click the action for which you want to change the key sequence, and then click Change Key Sequence.
  5. If necessary, select the check box to enable an action, and then click the key sequence you want to use. 
  6. Click OK four times.
